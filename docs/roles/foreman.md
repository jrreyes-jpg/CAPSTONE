# Foreman

## Primary Responsibility

The Foreman handles actual on-site asset usage tracking and field-level monitoring.

## Main Duties

- Track asset usage on site
- Scan QR codes for asset identification
- Log asset usage, return, or field notes
- Monitor on-site manpower and operational activity

## Allowed Actions

- Use the QR scanner for field asset tracking
- Record who used an asset and where it was used
- Add notes related to asset condition or location
- Support site progress reporting

## Not the Main Owner Of

- Master inventory setup
- Project creation
- Global user management
- Client project overview

## System Intent

This role is closest to field operations and should own real-world asset movement tracking.
