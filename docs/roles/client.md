# Client

## Primary Responsibility

The Client views project status only.

## Main Duties

- View assigned projects
- Check current project status
- Monitor project progress updates

## Allowed Actions

- View project summaries
- View project status and progress only

## Not the Main Owner Of

- Project creation
- Task assignment
- Task updates
- Inventory control
- Asset tracking
- User management

## System Intent

This role is view-only and should stay limited to project status monitoring.
