# Super Admin

## Primary Responsibility

The Super Admin creates and manages projects, assigns users, and manages the master inventory.

## Main Duties

- Create and manage projects
- Assign engineers, foremen, and clients
- Manage user accounts and access
- Maintain the master inventory and asset records

## Allowed Actions

- Create and manage projects
- Create user accounts
- Assign users to the correct workflow
- Manage inventory and assets
- Generate or print asset QR codes

## Not the Main Owner Of

- Daily task updates
- Actual on-site asset usage logging
- Client-side project viewing

## System Intent

This role is the control point of the platform and should handle setup, assignment, and master inventory only.
